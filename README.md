# Behavior-Based Category Encoding

A project page about encoding high-cardinality categories by historical behavior instead of raw identity.

## Project Website

[View Project Site](https://mah-trigui.github.io/behavior-based-category-encoding/)

## Overview

This project used target-based encodings for support-case metadata with many categorical levels.

Instead of representing a category only by identity, the encoding represented how that category historically behaved with respect to escalation.

## Core Idea

One-hot says what a category is.

Behavior-based encoding says what it tends to do.
